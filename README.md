# 📊 Dashboard UI (Header & Sidebar)

## 🚀 Overview

This project is a **React + Tailwind CSS implementation** of a dashboard layout focusing on:

* ✅ Header (Navbar)
* ✅ Sidebar Navigation

The UI is built based on a Figma design with attention to spacing, colors, and layout.

---

## 🛠 Tech Stack

* React (Vite)
* Tailwind CSS
* Lucide React (icons)

---

## ✨ Features

### 🔹 Header (Navbar)

* Gradient background (Primary & Secondary colors)
* Search bar with icon and shortcut (`⌘K`)
* Notification icon
* User avatar
* Workspace dropdown

### 🔹 Sidebar

* Sectioned navigation (My Projects, Orchestrator, Admin)
* Icons using Lucide React
* Active state highlight (Knowledge Base)
* Hover effects for menu items
